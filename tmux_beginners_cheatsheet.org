| <P> Prefix (C-b)          | Description                     |
|---------------------------+---------------------------------|
| <P> c                     | New window                      |
| <P> &                     | Close window                    |
| <P> n                     | Next window                     |
| <P> p                     | Previous window                 |
| <P> l                     | Last window                     |
| <P> 0..9                  | Select window by index          |
| <P> ,                     | Rename window                   |
| <P> %                     | Split horizontally 50%          |
| <P> "                     | Split vertically 50%            |
| <P> o                     | Cycle through panes             |
| <P> left arrow            | Switch to left pane             |
| <P> right arrow           | Switch to right pane            |
| <P> top arrow             | Switch to top pane              |
| <P> bottom arrow          | Switch to bottom pane           |
| <P> !                     | Convert pane to window          |
| <P> z                     | Zoom pane                       |
| <P> space bar             | Cycle through layouts           |
| <P> [                     | Enter copy mode                 |
| <P> ]                     | Paste buffer                    |
| <P> =                     | List buffers                    |
| <P> :resize -[L R U D] 10 | Resize pane in [direction] size |
| <P> (                     | Move to next session            |
| <P> )                     | Move to prev session            |
